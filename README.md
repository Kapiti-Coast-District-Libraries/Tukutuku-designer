# Tukutuku Studio

Tukutuku Studio is a digital design tool for creating traditional Māori tukutuku (latticework) patterns. It allows users to design intricate patterns using a variety of stitch styles and traditional colors, with features like vertical symmetry and high-quality SVG export.

## Features

- **Traditional Stitch Styles:** Choose from various styles including Woven, Curved X, Diamond, and more.
- **Traditional Color Palette:** Use colors inspired by traditional materials like harakeke (flax) and pingao (golden sand sedge).
- **Symmetry Tools:** Enable vertical mirror symmetry to create balanced, traditional designs.
- **Customizable Grid:** Adjust the number of rows and columns to fit your design needs.
- **High-Quality Export:** Export your designs as standard SVG vectors or as a "Stylized Export" complete with a frame and artwork details.

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/tukutuku-studio.git
   cd tukutuku-studio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

### Deployment

This project is ready to be deployed on platforms like **Vercel** or **Netlify**.

1. Push your code to a GitHub repository.
2. Connect your repository to Vercel.
3. Vercel will automatically detect the Vite configuration and deploy your site.

## Cultural Significance

Tukutuku is a traditional Māori art form of decorative latticework. These panels are typically used to decorate the interior walls of a wharenui (meeting house). The patterns are created by stitching onto a grid of vertical stalks (kaho) and horizontal slats (kakaho).

Each pattern carries deep symbolic meaning, reflecting Māori mythology, history, and the natural world.
